# Portal de Denúncias de Lixo Urbano


Site completo para registrar denúncias de lixo urbano com mapa interativo e painel administrativo.


## Funcionalidades
- Enviar denúncias com fotos
- Marcar localização no mapa
- Painel da prefeitura para visualizar denúncias
- Salvar tudo no Firebase


## Como usar
1. Substitua as chaves do Firebase em script.js
2. Abra index.html para enviar denúncias
3. Abra admin.html para visualizar denúncias
